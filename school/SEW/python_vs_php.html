Both PHP and Python


  • interpreted, high level languages with dynamic typing
  • easy to learn (compared to C++, Perl)
  • extremly portable - run on all platforms
  • have a standardized database API
  • support lambdas and other builtin functional prog. constructs
  • support namespaces
  • large developer communities

Strengths of PHP


  • an commonly installed environment
  • private, protected and public modifiers
  • abstract and final modifiers for classes and methods
  • interfaces
  • wide range of byte-code caches
  • embedding in HTML

Strengths of Python


  • indentation is used to mark out block structure
  • modules
  • keyword arguments (in comparison to PHP default arguments)
  • true object orientation
  • multiple inheritance
  • everything is a reference!
  • consistent case sensitivity
  • structured exception handling
  • general purpose tool
  • large, robust, maintainable code bases
